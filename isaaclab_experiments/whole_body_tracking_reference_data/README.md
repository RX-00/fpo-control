# Reference Motion Data

This directory stores `.npz` motion files used for whole-body motion tracking.

The files are automatically downloaded and generated by `setup_env.sh`, which runs
`download_lafan_data.py` to fetch LAFAN1 CSV data from HuggingFace and convert it
to NPZ format via IsaacSim forward kinematics.

## Motion Files (generated after setup)

| File | Description |
|------|-------------|
| `walk1_subject1.npz` | Walking |
| `run1_subject2.npz` | Running |
| `jumps1_subject1.npz` | Jumping |
| `dance1_subject1.npz` | Dancing (subject 1) |
| `dance1_subject2.npz` | Dancing (subject 2) |
| `fight1_subject2.npz` | Fighting |
| `fallAndGetUp1_subject1.npz` | Fall and get up |
